# 🏅 Sports Data Analysis using Excel

## 📌 Project Overview
This project is based on a real-world case study provided by Analytics Lab. The objective was to analyze sportsmen data and extract meaningful insights using Microsoft Excel.

The dataset contains information about players, sports, countries, gender distribution, and other attributes.

---

## 🎯 Objectives
- Analyze sports participation across different countries
- Understand gender distribution (Male vs Female athletes)
- Explore different sports categories (Indoor vs Outdoor)
- Generate insights from athlete-level data

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning Techniques
- Basic Data Analysis

---

## 📊 Steps Performed
- Cleaned and structured raw data
- Used multiple sheets (SPORTSMEN, SPORT, LOCATION)
- Created Pivot Tables for analysis
- Analyzed country-wise and gender-wise distribution
- Generated summary reports

---

## 📈 Key Insights
- Identified top countries with highest participation
- Compared male vs female athlete distribution
- Found trends in indoor vs outdoor sports
- Built structured analysis for decision-making

---

## 📁 Dataset Sheets
- SPORTSMEN: Athlete-level data
- SPORT: Sports categorization
- LOCATION: Country and language mapping
- ANALYSIS & REPORT: Final summarized insights
